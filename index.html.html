Summary: Latitude vs X

The purpose of this is to see how countries relation to the equator impacts various elements of weather. To obtain data for this analysis we utitilized the OpenWeatherMap API. Graphics were generated using Matplotlib.

By selecting different photos to the right you can see how a countries latitude impacts the various elements of max temperature, humidity, cloudiness, and wind speed. If you want to compare them all together feel free to select the comparison tab and to see the data utilized to generate these graphics view the data page which has navigation in the upper right hand corner.
